BOUNTY BOARD

This was a system I made primarily for RP servers which allowed certain jobs to place bounties on people. This system had a config that let the server owner edit which jobs could place, see, and redeem the bounties. When the list of bounties filled up it would slowly scroll through them, highlighting one person at a time.

THERMAL VISION

A TFA Attachment that darkens the screen and highlights visible NPCs and Players. Inspired by the one in GTA 5.

AI TURRET

This is an entity that automatically finds and shoots NPCs or Players with a settings menu to change its alliance, tracer, damage and health.

RICOCHET SHOT

This is a TFA Attachment that would bounce the shot off of walls or between targets with a laser to help find where the shot will land. If it hits a player it will bounce between nearby players, if it hits an NPC it will bounce between nearby NPCs.

REBORN SCOREBOARD

This is a scoreboard made for Reborn JvS, it also contains a menu when clicking on people to copy their steamid and see their money, playtime and staff role

SKIPRAY BLASTBOAT

This is an LFS ship that utilizes many different Garry's Mod systems. It has adaptive exhaust which will change size based on the ships throttle, a 360 degree turret which can be shot by the driver by holding alt and shooting or by a passenger, and the tail rotor can be swiveled 90 degrees like it does in lore.

STARCHASER

CLONE TURBO TANK

GUI-MENU LIBRARY

CUSTOM ITEM BASE

MONEY DROPS

This system made it so there is a chance for any mob to drop a money bag with a min and max value defined in a JSON config.

This system allowed the player to talk to NPCs to earn and complete quests. I utilized a MySQL database to more efficiently store the questing data.

This system included the Custom Mobs and allowed me to easily create arenas and bosses to go with them. The arenas need an origin point, radius, and list of bosses that will spawn and it will create the arena and boss spawner block/menu to go along with it.
